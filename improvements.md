# שינויים לעומת הסקר המקורי (תשפ"ו-2025)

## עדכוני גרסה 2 (לאחר משוב)
- **טון**: הפתיח קוצר לשתי שורות; כל ההסברים עברו לחלונית שו"ת (❓ בפינה). הוסרו התנצלויות והסברי-יתר מהשאלות.
- **מסך ראשון**: במקום 39 פרקים — **6 תחומים** (ניהול/תשתיות/מרחב ציבורי/חינוך/קהילה/תרבות), כל אחד עם צבע וסרגל התקדמות משלו.
- **התחלה מיידית**: לוחצים "מתחילים" ← בוחרים תחום ← שאלות הפילוח מופיעות פעם אחת בלבד, רגע לפני הפרק הראשון.
- **שם פרטי + משפחה** אוחדו לשדה "שם מלא" אחד.
- **המשך ממכשיר אחר**: בוטל קוד ההמשך. במקומו — כפתור "אמשיך אחר־כך" קבוע: משאירים טלפון, ובמכשיר השני מזינים אותו. בגיליון נשמר hash בלבד, לא המספר.
- **המשך אוטומטי**: בסיום פרק עוברים ישר לנושא הבא בתחום — בלי לחזור לרשימה.


כל השאלות נשמרו בתוכנן. להלן תיעוד מלא של מה ששונה — לביקורת הצוות.

## שינויים מבניים

1. **פרק חדש "קצת עליך"** בתחילת הסקר: מגדר, גיל (בטווחים), ותק (בטווחים), סוג מגורים, ילדים בבית, חברות באגודה. משמש לפילוח **וגם ללוגיקת הדילוג** — במקום 6 שאלות סינון שהיו מפוזרות בסקר.
2. **שאלות סינון שהוסרו** (מוחלפות בפרופיל):
   - "האם אתה תושב שכונות הקראוונים?" + "באיזו שכונת קראוונים אתה גר?" → סוג מגורים בפרופיל
   - "האם אתה גבר?" (במקווה) → מגדר בפרופיל (הפרק מוסתר מגברים)
   - "האם יש לך בני נוער (ז׳–י״ב)?" (בנוער) → ילדים בפרופיל
   - "כמה שנים אתה גר ביישוב?" (טקסט חופשי) → טווחי ותק בפרופיל
3. **שאלות "מעוניין בכל זאת לענות על פרק X?" הוסרו** (תקציב, מועצה) — במערכת החדשה מסך הפרקים עצמו הוא הבחירה; מי שלא רוצה פשוט לא נכנס לפרק.
4. **טבלת הדירוג של נושאי הקהילה** (9 נושאים בגריד 9×9) → רשימת **גרירה לסידור** — פשוטה בהרבה במובייל, ומונעת מובנית דירוג כפול.
5. **שאלה שאוחדה**: "האם אתה מרוצה מתהליך הקליטה?" (טקסט פתוח) אוחדה עם "מה אפשר לשפר בתהליך הקליטה?" — היו כפולות.
6. **"עד כמה את/ה מרגיש נגישות ופשטות בקבלת עזרה"** (עזרה הדדית) — הייתה שאלת טקסט פתוח, הומרה לסולם 1–10 (כפי שמשתמע מניסוחה) + נשארה שאלת הטקסט הפתוח שאחריה.
7. **טווחי הקליטה תוקנו**: במקור היו חופפים (2–3, 4–6, 6–8, 8–10…) — עכשיו רציפים: 2–3, 4–6, 7–10, 11–15, 15+.
8. **תנאי הצגה חדשים בתוך פרקים** (לא היו במקור כי גוגל פורמס מוגבל): שאלות "עד כמה אתה מרוצה מ-X" מוצגות רק למי שענה שהוא משתמש ב-X (בית כנסת, ספרייה, מועדון, משחקייה, מקווה, תרבות). מפחית רעש בתוצאות.
9. **"האם תשקול להצטרף כמנוי (בהנחה של 5 ₪)?"** מוצגת רק למי שאינו מנוי.

## תיקוני שגיאות כתיב וניסוח מהמקור

| מקור | תוקן |
|---|---|
| "האם אתה מגיש שיפור ברמת השירות" | מרגיש |
| "האם אתה מכיר באפשרויות השירות" | את אפשרויות |
| "רצת בטון בגודל בסיסי" | רצפת בטון |
| "הם אתה משתמש בשירותי בית הכנסת" | האם |
| "מה הסיבות שבגינם אינך משתמשת במ'קווה" | מהן הסיבות… במקווה |
| "כן מגוון מפסיק" | מגוון מאוד (תווית סולם) |
| "וכד'ו" | הוסר לניסוח נקי |
| "שלושה קבוצות" | שלוש קבוצות |
| "תחושה גבוה", "מעורבות גבוה" וכד' | גבוהה |
| "באיזה תדירות" | באיזו תדירות |
| "עד כמה תעדיפו" (ערבוב יחיד/רבים) | אוחד ליחיד |
| תוויות קצה "כן מרוצה" | "מרוצה מאוד" (ברור יותר בקצה סולם) |

## מה נשמר בכוונה

- הניסוח בלשון זכר + ההערה בפתיח (כמו במקור); פרק המקווה בלשון נקבה.
- ההומור: "מאלף אריות אסייתיים" בפאב 🦁
- כל האפשרויות, הסולמות (1–10) ותוויות הקצה — בתוכן זהה.
- סדר הפרקים לפי סדר התקציב.
